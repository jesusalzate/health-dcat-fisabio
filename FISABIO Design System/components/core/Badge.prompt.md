A compact pill label for categories, statuses and tags (e.g. research areas, project states). Soft variants for neutral tagging; `success`/`warning`/`danger` for status.

```jsx
<Badge variant="soft">Epidemiologia</Badge>
<Badge variant="coral">Nou</Badge>
<Badge variant="success" dot>Actiu</Badge>
```

Variants: `solid` `accent` `soft` `sky` `coral` `success` `warning` `danger` `outline`. Add `dot` for a status indicator.
