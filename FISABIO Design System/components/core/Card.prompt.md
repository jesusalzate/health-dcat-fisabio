The standard FISABIO surface — soft indigo-tinted shadow, generous radius. Use `accent` for the short coral bar on feature cards, `brand` for indigo call-out cards, `interactive` for clickable items in a grid.

```jsx
<Card title="Epidemiologia i Salut Pública" accent interactive>
  Vigilància i resposta davant amenaces per a la salut de la població.
</Card>

<Card brand>
  <h3>Memòria d'activitat 2025</h3>
  <p>Consulta els resultats de l'any.</p>
</Card>
```

Props: `title`, `accent`, `brand`, `interactive`, `flat`, `padded`.
