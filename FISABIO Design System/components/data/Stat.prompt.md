```jsx
<Stat value="2,847" label="Publicacions indexades" desc="Articles 2020–2025" />
<Stat value="42" suffix="M€" tone="accent" label="Finançament captat" />
<Stat value="3" suffix="províncies" size="sm" />
```

Props: `value`, `suffix`, `label`, `desc`, `tone` (`brand`/`accent`/`onbrand`), `size`.
