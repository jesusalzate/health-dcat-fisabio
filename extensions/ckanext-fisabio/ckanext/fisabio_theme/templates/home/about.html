    Acerca de

    Un portal para descubrir y reutilizar datos de investigación sanitaria

    El Portal de Datos Abiertos de FISABIO centraliza la descripción, búsqueda y acceso a conjuntos de datos vinculados a investigación sanitaria y biomédica. Su objetivo es facilitar el descubrimiento de recursos, mejorar la trazabilidad y apoyar un uso responsable de los datos.

    Qué ofrece

    Un catálogo con metadatos, recursos, formatos, organizaciones y filtros para localizar información relevante.

    Para quién es

    Equipos investigadores, personal técnico, gestores de datos y entidades colaboradoras que necesitan localizar o solicitar datos.

    Cómo se accede

    Los datos abiertos pueden consultarse desde el catálogo; los restringidos requieren una solicitud de acceso y revisión previa.

    Tecnología

    Construido sobre CKAN

    Este portal utiliza CKAN, una plataforma de código abierto especializada en catálogos de datos. CKAN permite gestionar metadatos, organizaciones, recursos descargables, formatos, búsquedas, filtros y acceso mediante API, manteniendo una base técnica sólida para publicar y gobernar datos de investigación.
