    Espacio de datos FISABIO

    Datos sanitarios preparados para investigar con rigor

    El espacio de datos de FISABIO organiza conjuntos de datos sanitarios y biomédicos para facilitar su descubrimiento, consulta y reutilización bajo criterios de calidad, interoperabilidad, trazabilidad y gobernanza responsable.

    Explorar catálogo Solicitar acceso a datos

    Descubrimiento

    El catálogo reúne metadatos, formatos, organizaciones y etiquetas para encontrar datos relevantes con rapidez.

    02

    Gobernanza

    Los datos se publican con controles de acceso, criterios de uso y procedimientos de solicitud cuando el contenido lo requiere.

    03

    Interoperabilidad

    La estructura CKAN permite documentar recursos, formatos y vocabularios para favorecer la reutilización técnica y científica.

    Acceso a datos

    El formulario de solicitud no es el inicio de sesión

    El botón “Solicitar acceso a datos” abre un formulario público para pedir acceso a conjuntos de datos restringidos. El equipo administrador revisa la solicitud y contacta por correo electrónico. El inicio de sesión, en cambio, es para usuarios internos, administradores o personas con cuenta CKAN.

    Cómo funciona

    Del dato al uso investigador

    1. PublicarSe documentan conjuntos de datos y recursos asociados.
    2. EncontrarEl catálogo permite buscar, filtrar y comparar información disponible.
    3. SolicitarCuando procede, el acceso se gestiona mediante un formulario y revisión administrativa.
